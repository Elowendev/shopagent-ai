# 小拾光 Agent · ShopAgent AI

## 项目定位

小拾光 Agent（ShopAgent AI）是一个面向原创品牌店铺的 AI 客服 Agent 系统。

它不是普通聊天机器人。

而是：

一个能够理解品牌风格、商品语义、用户需求，并自主生成售前与售后回复的 AI 店员系统。

当前品牌案例：

Aurora Whisper

主营：

* 原创手工饰品
* 天然石饰品
* 项链 / 手链
* 面向年轻女生

---

# 项目目标

构建一个：

“真正可以服务真实店铺”的 AI Agent 系统。

而不是：

* Demo
* Prompt 展示
* 套壳聊天机器人

---

# 当前阶段（MVP）

当前阶段：

只开发最小可用版本（MVP）。

目标：

* AI 聊天界面
* 商品知识库
* AI 商品推荐
* 品牌语气生成

# 核心能力

## 1. 商品推荐

AI 能根据：

* 用户描述
* 穿搭风格
* 情绪关键词
* 场景需求

推荐适合的饰品。

---

## 2. 品牌语气

AI 回复必须符合：

Aurora Whisper 的品牌风格：

* 温柔
* 安静
* 低饱和
* 自然感
* 日常感

禁止：

* 淘宝客服语气
* 夸张营销
* “亲亲”“宝子”等表达

---

## 3. 售后回复

支持：

* 发货问题
* 材质问题
* 保养问题
* 尺寸问题
* 是否掉色
* 是否适合送礼

---

## 4. Agent 决策能力

AI 必须具备：

* 用户意图识别
* 商品检索
* FAQ 查询
* 回复生成

未来支持：

* Tool Calling
* Memory
* 多步骤推理
